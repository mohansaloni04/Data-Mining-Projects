2. Then install flask on it and import all the libraries need for the integration like numpy , panda, bs4, scipy etc

3. We made a python file called process.py which render the template when button is clicked 

4. Process.py also calls the backend to process the backend

5. Form.html gives the homepage for the project

6. Output.html provides the output of the product rating
